# crzutils

crzutils combines some utilities into a single small executable.

## Installation

```sh
make
make install
```

## Usage

Currently defined commands:

- `apt-du`
- `bye`
- `droidmnt`
- `extract`
- `git-cloc`
- `nsa-name`
- `openw`
- `own`
- `search`
- `start`
- `vimv`
- `xid`
- `yt-music`

See `[COMMAND] --help` for info on usage.
